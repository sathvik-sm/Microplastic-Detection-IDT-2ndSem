# Microplastic Detection Dashboard

## Overview

The Microplastic Detection Dashboard is a web-based monitoring system developed as part of an Interdisciplinary Technical (IDT) project. The dashboard displays sensor measurements, calculated capacitance values, phase information, estimated microplastic concentration, and real-time signal waveforms.

The application provides a user-friendly interface for visualizing data collected from the sensing hardware and updates automatically as new measurements become available.

---

## Features

* Real-time dashboard updates every 500 ms
* Live display of:

  * Capacitance (F)
  * Phase (degrees)
  * Alpha value
  * Estimated microplastic concentration
  * Timestamp of latest measurement
* Separate real-time graphs for:

  * Input Signal (Vin)
  * Output Signal (Vout)
* Responsive web interface
* Data stored and retrieved using JSON files
* Dynamic graph rendering using Chart.js

## How It Works

1. Sensor measurements are generated by the hardware system.
2. Processed values are written into `data.json`.
3. The Express server reads the data and serves it to the dashboard.
4. The dashboard periodically requests updated data from the `/api/data` endpoint.
5. Graphs and dashboard metrics are refreshed automatically without reloading the page.
